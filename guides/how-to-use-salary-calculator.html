工资计算器怎么用？税前工资、五险一金、个税到手金额计算教程

本工具适配2026年现行个税与社保政策，支持正向测算到手工资、反向反推税前薪资，可自定义全部扣除参数，精准匹配跳槽谈薪、offer核算、年终奖测算等场景。以下是核心功能说明与分场景操作步骤。

一、核心功能亮点

  1. 双向计算，覆盖全场景

    同时支持「税前工资推算到手工资」和「目标到手工资反推税前薪资」，既可以拿到offer后核算实际收入，也能在谈薪前确定报价底线，覆盖求职全流程需求。

  2. 全参数自定义，贴合实际情况

    可自由设置社保公积金缴纳基数、各项缴纳比例，支持自定义7项专项附加扣除，匹配不同城市、不同公司的薪资缴纳标准，避免默认参数带来的计算误差。

  3. 累计预扣法精准计税，输出全年明细

    严格按照现行累计预扣规则计算，自动识别税率跳档节点，输出12个月每月到手金额、个税扣除明细，直观呈现全年薪资变化，解决单月计算不准的问题。

  4. 年终奖计税对比与优化筹划

    支持单独计税、并入综合所得两种方式自动测算，直接显示到手差额，自动提示税率临界点；可搭配年终奖分配优化计算器，通过合理拆分月薪与年终奖的金额组合，测算全年最低综合税负方案。

  5. 政策同步更新，无需手动核对

    同步当前个税起征点、七级累进税率表及社保通用规则，参数默认值符合主流城市标准，无需自行查询政策变动，直接使用即可。

二、分场景操作步骤

场景一：税前工资算到手（拿到offer核算实际收入）

适用于已收到offer，想知道税前工资实际每月到手多少、全年总收入多少的场景。

  1. 选择「税前→到手」计算模式
  2. 输入税前月薪金额
  3. 按实际情况设置社保公积金参数：填写缴纳基数、各项缴纳比例
  4. 勾选并填写符合条件的专项附加扣除项目
  5. 点击计算，即可查看每月到手、个税明细及全年汇总数据

提示：社保公积金基数请以offer或公司实际缴纳标准为准，基数不同，到手金额差异会比较明显。

场景二：到手工资反推税前（跳槽谈薪确定报价）

适用于有目标到手金额，想反推出需要谈多少税前工资的谈薪场景。

  1. 切换至「到手→税前」反推模式
  2. 输入目标每月到手工资金额
  3. 设置社保公积金缴纳基数、比例及专项附加扣除参数
  4. 点击计算，即可得出对应所需的税前月薪，附带全年薪资明细

场景三：年终奖计税测算与优化

适用于核算年终奖税后金额、选择最优计税方式的场景。

  1. 完成基础薪资参数设置后，勾选「年终奖计算」选项
  2. 输入年终奖金额，或选择按N个月工资折算
  3. 点击计算，自动生成两种计税方式的应缴个税与到手金额，标注最优方案
  4. 如需进一步优化全年整体税负，可使用年终奖分配优化计算器做精细化拆分测算

所有计算结果均基于输入参数与现行政策得出，可作为谈薪、核对工资的参考，实际发放金额以公司薪资核算为准。
