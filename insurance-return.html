工具介绍

本工具是一款免费的在线保险收益分析计算器，帮助您快速评估保险产品的真实收益水平。支持增额终身寿、年金险、万能险、分红险、两全保险等多种保险类型。只需粘贴保险合同中的现金价值表，即可自动计算内部收益率（IRR），并与银行定期存款进行对比分析。

如何使用

  1. 选择保险类型：在下拉菜单中选择您购买的保险类型（增额终身寿、年金险等）。
  2. 填写基本信息：输入年缴保费、缴费年限，投保年龄为选填项。
  3. 录入现金价值：从保险合同中复制「现金价值表」数据，粘贴到文本框中点击「解析数据」；或切换到「手动录入」模式，填写关键年份的现金价值。
  4. 录入减保信息（可选）：如计划在持有期间部分领取现金价值，可添加减保记录。
  5. 设置对比参数：输入银行定期存款利率和预期通胀率，用于收益对比分析。
  6. 点击「开始分析」：系统将自动计算 IRR、回本年限、与银行存款的对比结果，并生成可视化图表。

计算原理

本工具采用 IRR（内部收益率） 作为核心指标，IRR 是衡量投资真实收益水平的年化复合收益率，已充分考虑资金占用的时间价值。

计算公式：通过迭代法求解使得所有现金流净现值（NPV）等于零的折现率 r，即满足：

Σ [CFt / (1+r)t] = 0

其中 CFt 为第 t 年的现金流（保费支出为负，退保/满期金收入为正）。系统同时计算名义 IRR（未扣除通胀）和真实购买力 IRR（扣除通胀后），并与同期银行定期存款复利收益进行对比。
